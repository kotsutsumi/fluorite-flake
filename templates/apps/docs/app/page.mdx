## 💡 よくある質問

### 環境変数はどこに書く？

各アプリの `.env.local` ファイルに記載します。初期生成は `pnpm env:init` で実行できます。

詳細は [環境変数管理](/setup/environment-variables) を参照してください。

### データベースのマイグレーションはどうする？

ローカル開発では `pnpm db:reset`、本番環境では `pnpm db:cloud:migrate` を使用します。

詳細は [データベース操作ガイド](/guides/database-operations) を参照してください。

### チームで環境変数を共有するには？

`pnpm env:encrypt` で暗号化し、`pnpm env:decrypt` で復号します。

詳細は [チーム開発ガイド](/guides/team-collaboration) を参照してください。

## 🛠️ プロジェクト構成

このモノレポは以下のアプリとパッケージで構成されています：

### Apps

- **web**: マーケティングサイト・フロントエンド（ポート 3000）
- **backend**: 管理ダッシュボード + API サーバー（ポート 3001）
- **docs**: ドキュメントサイト（ポート 3002）
- **mobile**: Expo モバイルアプリ（ポート 8081）

### Packages

- **ui**: 共通 UI コンポーネントライブラリ（shadcn/ui ベース）
- **typescript-config**: 共通 TypeScript 設定
