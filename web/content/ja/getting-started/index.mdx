# クイックスタート

まずは CLI をインストールし、テンプレートからプロジェクトを生成して動作を確認します。

## 必要条件

- Node.js 20 以上
- pnpm 8 以上 (`corepack enable` でバージョンを固定することを推奨)

以下のコマンドで環境を確認します。

```bash
node --version
pnpm --version
```

## インストール

プロジェクトディレクトリで `pnpm dlx` を使用して CLI を実行できます。グローバルインストールは不要です。

```bash
pnpm dlx fluorite-flake create my-app
```

CLI は対話形式でテンプレートの種類、パッケージマネージャー、追加機能を選択できるので、プロンプトに従って設定を完了してください。

## 開発フローの確認

プロジェクト生成後は README に記載された手順で依存関係のインストールと開発サーバーの起動を行います。Next.js テンプレートの場合の一例:

```bash
cd my-app
pnpm install
pnpm dev
```

CLI のデバッグモードを利用したい場合は以下を参考にしてください。

```bash
FLUORITE_DEBUG=true pnpm dlx fluorite-flake create sample
```

## 次のステップ

- CLI のオプション詳細は「CLI リファレンス」を参照
- テンプレートの構成やカスタマイズ方法は Fluorite Flake リポジトリの `templates/` ディレクトリをチェック
- 社内テンプレートを追加する場合は `src/commands/create` 配下のロジックを拡張し、テストを追加してください
