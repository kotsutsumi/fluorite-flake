# Get Started

## Installation

### Global Installation

Install Fluorite-flake globally to use it from anywhere:

```bash
npm install -g fluorite-flake
```

### Local Installation

You can also install it locally in your project:

```bash
npm install fluorite-flake --save-dev
```

## Creating Your First Project

<Steps>

### Run the Create Command

```bash
fluorite-flake create
```

### Choose Your Framework

Select from the available options:
- **Next.js**: Modern React framework with TypeScript
- **Expo**: React Native for mobile development
- **Tauri**: Cross-platform desktop apps with Rust

### Configure Your Project

Choose additional features:
- **Database**: Turso, Supabase, or SQLite
- **Authentication**: Framework-specific auth solutions
- **Storage**: Cloud storage providers
- **Testing**: Comprehensive test setup

### Start Development

Navigate to your project and start developing:

```bash
cd your-project-name
npm run dev
```

</Steps>

## Dashboard Interface

Launch the interactive TUI dashboard for project management:

```bash
fluorite-flake dashboard
```

The dashboard provides:
- **Project Overview**: Current project status
- **Quick Actions**: Common development tasks
- **Configuration**: Easy settings management
- **Monitoring**: Real-time project metrics

## Available Commands

| Command | Description |
|---------|-------------|
| `create` | Create a new project |
| `dashboard` | Launch TUI dashboard |
| `--help` | Show help information |
| `--version` | Show version information |
