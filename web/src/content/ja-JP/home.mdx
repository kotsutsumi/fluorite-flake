## 🚀 クイックスタート

```bash
# グローバルインストール
npm install -g fluorite-flake

# 新しいプロジェクトを作成
fluorite-flake create

# ダッシュボードを起動（ベータ）
fluorite-flake dashboard
```

「s」キーでVercelとTursoを切り替え、「q」またはEscで終了できます。
