# Fluorite-flake 🚀

**Fluorite-flake** is a powerful multi-framework project generator and CLI utility that creates production-ready applications with modern best practices, comprehensive testing, mobile support, and integrated documentation.

## ✨ Key Features

- **🎯 Multi-Framework Support**: Create projects with Next.js, Expo, or Tauri
- **📱 Mobile Testing**: Built-in Maestro E2E testing for Expo projects
- **🗄️ Flexible Database Options**: Choose between Turso, Supabase, or local SQLite
- **🔐 Authentication Ready**: Framework-specific auth solutions
- **☁️ Storage Solutions**: Multiple storage providers support
- **🧪 Comprehensive Testing**: 32 test files covering unit, integration, and E2E testing
- **🌍 Internationalization**: Native support for English and Japanese

## Quick Start

```bash
# Install globally
npm install -g fluorite-flake

# Create a new project
fluorite-flake create

# Launch TUI dashboard
fluorite-flake dashboard
```
